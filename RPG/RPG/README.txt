#####################################
MOVEMENT Controls
#####################################

W - MOVE NORTH
S - MOVE SOUTH
A - MOVE WEST
D - MOVE EAST

#####################################
INVENTORY Controls
#####################################
1 - Equip HEAD Inventory SLOT if there's an item that fits here.
2 - Equip CHEST Inventory SLOT if there's an item that fits here.
3 - Equip ARMS Inventory SLOT if there's an item that fits here.
4 - Equip LEGS Inventory SLOT if there's an item that fits here.
5 - Use Potion of Fire Resistance if there's an item in the bag.
6 - Use Potion of Water Resistance if there's an item in the bag.

#####################################
GAME STATE Controls
#####################################
U - RECORD MODE
Y - EXIT RECORD MODE
G - FWD PLAYBACK OF RECORDED STATES
G - REV PLAYBACK OF RECORDED STATES
P - PAUSE GAME
